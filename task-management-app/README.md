# TASK MANAGEMENT APP

The Project is Built on React

## Tech Stack Used

* HTML
* JavaScript
* React UI Library - Chakra UI
* Socket.io

### `npm start`

To Run the App. Move task-mangement-app folder. Use Command

```
npm start
```

### Features

1. Signup and Login
2. Create Task
3. Edit and update only by User
4. Admin can Also update task
5. Logout
